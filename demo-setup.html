🖼️ Configuration des Images

Le site utilise actuellement des images de démonstration depuis Unsplash. Pour utiliser vos propres images :

  1. Ajoutez vos images dans le dossier images/
  2. Modifiez les URLs dans js/products.js
  3. Optimisez vos images pour le web (format WebP recommandé)

⚙️ Personnalisation

Pour personnaliser votre boutique :

  1. Produits : Modifiez js/products.js
  2. Couleurs : Changez les variables CSS dans css/style.css
  3. Informations : Utilisez le panneau d'administration
  4. Logo : Remplacez le texte "RANIA SHOP" par votre logo

📱 Test sur Mobile

Le site est entièrement responsive. Testez-le sur :

  • 📱 Smartphones (320px - 767px)
  • 📱 Tablettes (768px - 1199px)
  • 💻 Desktop (1200px+)

🔧 Serveur Local (Recommandé)

Pour une meilleure expérience, utilisez un serveur local :

# Avec Python 3 python -m http.server 8000 # Avec Node.js (npx) npx http-server # Avec PHP php -S localhost:8000

Puis ouvrez : http://localhost:8000

🚀 Mise en Production

Pour déployer votre site :

  1. Changez les identifiants d'administration
  2. Remplacez les images de démonstration
  3. Configurez un serveur web (Apache/Nginx)
  4. Activez HTTPS
  5. Implémentez une base de données
  6. Ajoutez un système de paiement réel
